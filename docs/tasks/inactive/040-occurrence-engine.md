# 040 기간·반복 발생 일정 엔진

## 상태
- 상태: 비활성
- 브랜치: `feat/occurrence-engine`

## 목표
한 개의 일정 원본에서 조회 범위에 해당하는 단일·기간·반복 날짜 표시를 결정론적으로
생성하는 도메인 엔진을 만든다.

## 의존성
- 작업 030 완료

## 범위
- `CalendarOccurrence`와 bounded range 조회를 구현한다.
- 단일 날짜와 시작·종료일을 포함하는 연속 기간을 지원한다.
- 매일, 매주 요일, 매월 일자, 매년 월일 반복과 간격·종료일·무기한을 지원한다.
- 달력 조회를 발생 일정 기반으로 전환한다.

## 제외 범위
- 기간과 반복의 결합, 반복 횟수, 매월 n번째 요일은 지원하지 않는다.
- 날짜별 예외 인스턴스와 날짜별 완료 상태는 만들지 않는다.

## 설계 결정
- 기간과 반복 모드는 서로 배타적이다.
- 수정·완료·삭제는 항상 전체 시리즈 원본에 적용한다.
- 매월 존재하지 않는 29~31일은 건너뛰고 2월 29일 연간 반복은 윤년에만 만든다.
- 반복 종료일은 포함 범위로 해석한다.

## 완료 조건
- 같은 입력과 조회 범위는 항상 같은 발생 일정 집합을 반환한다.
- 무기한 반복도 유한한 조회 범위 안에서만 계산된다.

## 검증
- 기간 경계, 주간 요일, 간격, 월말, 윤년과 종료일 단위 테스트를 추가한다.
- 전체 시리즈 수정·완료·삭제와 달력 표시 회귀 테스트를 실행한다.
