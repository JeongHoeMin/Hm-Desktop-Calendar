# 160 주 시작 요일과 주말 숫자 색

## 상태

- 상태: 활성
- 브랜치: `feat/calendar-display-options`

## 목표

주 시작 요일(일요일/월요일)을 선택할 수 있게 하고, 주말 날짜 숫자에 요일
색(일요일 빨강, 토요일 파랑)을 입혀 달력의 완성도를 높인다.

## 의존성

- 140 (설정 저장소), 150 (설정 창 배치)
- 130 (소프트 의존: 날짜 숫자 색 우선순위 규칙 공유)

## 범위

- `WeekStart` 설정(일요일 기본/월요일)을 추가하고 설정 창에 선택 UI를 둔다.
- `CalendarViewModel`의 월 그리드 생성과 요일 헤더 순서를 주 시작 요일로
  매개변수화한다.
- 주말 날짜 숫자 색을 적용한다. 색은 요일 헤더에서 쓰는 기존 상수(일요일
  `#FF5065`, 토요일 파랑)를 재사용하고, 켜기/끄기 토글 1개를 둔다.
- 설정 변경 시 `CalendarViewModel.RefreshAsync`로 즉시 반영한다.

## 제외 범위

- ISO 주번호 표시.
- 주말 정의 커스터마이즈(토·일 고정).

## 설계 결정

- 날짜 숫자 색 우선순위를 130의 규칙과 단일 함수로 통합한다: 사용자 배경색
  밝기 기반 전경 > 공휴일 빨강 > 주말 색 > 테마 기본.
- 130이 아직 병합되지 않은 상태로 착수하면 주말 색 우선순위 규칙만 먼저
  구현하고, 130 병합 시 같은 함수에 공휴일 규칙을 합류시킨다.
- 앱 설정 스키마를 2로 올리고 기존 스키마 1 문서는 일요일 시작·주말 색 켜짐을
  기본값으로 읽은 뒤 현재 스키마로 정규화한다.

## 완료 조건

- 월요일 시작 선택 시 그리드, 요일 헤더, 이전/다음 달 채움 셀이 모두
  일관되게 정렬된다.
- 공휴일과 토요일이 겹치는 날은 공휴일 색이 우선한다.
- 설정이 재시작 후 유지된다.
